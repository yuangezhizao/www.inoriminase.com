    【しあわせ いのりまち】会員限定イベント グッズ販売決定！

    • ツイート
    水瀬いのり公式モバイルサイト「しあわせ いのりまち」の会員限定イベント、
    「第三回町民集会～ちょっと早めのまちの夏祭り～」のグッズのラインナップを大公開！
    定番アイテムからいのりまちで制作するのは初となるアイテムまで、
    どれを手にとっても夏気分を先取りしていただけること間違いなし！
    今回のために撮り下ろしたビジュアル、描き下ろしたイラストも必見です！
     
    ▼ラインナップ・詳細はこちら
    http://smam.jp/inori_vol3/
     
    【先行販売予定時刻】
    ●NHK大阪ホール     6月10日(土) 12:00
    ●品川ステラボール  ６月17日(土) 11:00
     
    【グッズ購入制限について】
    おひとり様一会計につき、1アイテム2個までとさせていただきます。
     
    【グッズ販売に関するご注意】
    ●ご購入の際に「しあわせ いのりまち」会員様認証を行います。住民票(マイページ)のご提示が必要となりますので、必ずご用意ください。
    ●先行販売時間内には、公演チケットをお持ちでない会員様も上記のご提示でご購入が可能となります。
    ●準備の状況により、販売開始時刻が前後する場合がございます。
    ●表示価格はすべて消費税込です。
    ●グッズをお買い求めの際は、その場で、個数・合計金額・商品の不備等をご確認ください。後日の交換対応は原則として致しかねますので、あらかじめご了承ください。
    ●イベント前日、及び当日の会場・会場周辺での徹夜、早朝からの座り込みはご遠慮ください。会場周辺の方のご迷惑になるとともに、お客様の安全を守る為、徹夜行為は一切禁止させていただきます。上記が守られない場合、状況によりイベントやグッズ販売を中止する場合もございますので十分ご注意ください。
    ●販売当日は大変混雑が予想されます。周囲の方のご迷惑になるような行為はご遠慮ください。また、会場係員の誘導には必ず従ってください。
    ●商品の画像は実際の色味やデザインとは異なる場合がございます。
     
    【クレジットカードのご利用について】
    ●一部の窓口でクレジット・デビット・プリペイドカードのご利用が可能となります。
    ●クレジットカードのお支払い回数は「一括」のみとなります。
    ●クレジットカードはVISA,MasterCard,JCB,AMERICAN EXPRESS,Diners,
    DISCOVER,MUFG CARD,DC,NICOS,OMCがご利用いただけます。
    ●デビットカードはVISAデビット、JCBデビットがご利用いただけます。（※J-Debit はご利用いただけません。）
    ●プリペイドカードは VISA、MasterCard、JCBがご利用いただけます。（※ネット専用カードはご利用いただけません。）
    ●決済システムの都合により、デビットカードやプリペイドカードをご利用いただいた場合でも、明細には「クレジットカード決済」と表記される場合がございます。
    ●通信状況により手続きにお時間をいただく場合がございます。ご迷惑おかけ致しますが、ご理解・ご了承のうえご利用頂きますよう、お願い致します。
    ●お会計済みの商品との合算は承れません。
    ●全ての売場窓口での対応は致しておりません。クレジット対応の窓口をご案内致しますので、売場では係員の誘導に従ってください。待機列は同じになります。
