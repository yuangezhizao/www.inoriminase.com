    「Inori Minase LIVE TOUR 2019 Catch the Rainbow！」ライブグッズ通信販売スケジュール決定！

    • ツイート
    先日お知らせいたしました「Inori Minase LIVE TOUR 2019 Catch the Rainbow！」グッズ通信販売に関しまして、
    下記のサイトおよび期間で予約受付（受注販売）をさせていただきます。

    【キンクリ堂　通信販売特設サイト】
    http://kinkurido.jp/shop/pages/ctr-goods.aspx


    【予約受付期間】
    2019年7月3日(水) 午後7時00分～2019年7月12日(金) 午前11時59分
     
    【商品について】
    ※商品価格は全て消費税込になります。
    ※商品の画像は実際の色味やデザインとは異なる場合がございます。あらかじめご了承ください。
    ※「会場限定ラバーバンド」の取り扱いはございません。また、一部グッズは数量限定となります。
    ※事後販売では「水瀬いのり 特製トレカ“MINACA”」は付きません。
     
    【お支払い方法について】
    クレジット決済、コンビニ決済がご利用できます。
    ◆クレジット決済◆
    ※VISA・Master・ダイナース・アメリカンエキスプレス・JCBのマークの入っているクレジットカードをご利用になれます。
    ◆コンビニ決済◆
    ※コンビニ決済をご指定いただきますと、ご注文日より7日以内でのお支払いとなり、期日までにご入金いただけなかった場合はキャンセルとなります。
    ※コンビニ決済の際、各コンビニ一律185円手数料としていただいております。予めご了承ください。
     
    【配送について】
    ※商品のお届けは“9月下旬”頃を予定しております。
    ※お届け日が確定しましたらメールにてお知らせさせていただきます。
    ※商品の配達は佐川急便でのお届けとなり、送料￥540（税込）になります。
    ※商品代金￥10,800（税込）以上お買い上げいただきますと送料無料となります。
     
    【ご注文について】
    ※注文後のキャンセルはお受けできませんので予めご了承ください。
    ※ご購入にはキンクリ堂への会員登録（無料）が必要となります。
    ※「Inori Minase LIVE TOUR 2019 Catch the Rainbow！」グッズはその他の商品と一緒に注文はできません。
    ※ご注文完了のお知らせやコンビニお支払に必要となるお客様番号等はメールで送信されますのでメールアドレス（kingrecords.co.jp）を受信できるよう予め設定をお願いいたします。
    ※ご注文が完了した後、数分以内に｢ご注文の確認（自動配信メール）｣が必ず送信されます。メールが届かない場合はお手数ですがスマートフォン・携帯電話のメール受信設定を変更（「kingrecords.co.jp」をドメイン指定受信設定）し、キンクリ堂お問い合わせフォームよりご連絡ください。
